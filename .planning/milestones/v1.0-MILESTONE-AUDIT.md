## Tech Debt

### Phase 3: Native & Faces

1. **Codec dead code** — `RustJsonCodec`, `OrjsonCodec`, and `get_codec()` factory are built, parity-tested, and correct, but never wired into `HttpTransport` production path. HttpTransport uses `orjson` directly. The codec abstraction exists for future use.

2. **SchemaResolutionError unhandled** — When all cascade sources fail, `SchemaResolutionError` propagates as HTTP 500 / Python traceback through inbound adapters instead of a clean "schema unavailable" response.

3. **HttpTransport resource leak** — `close()` method exists but is never called. No context manager or `atexit` handler. Minor concern for long-running FastAPI; harmless for CLI/MCP.

4. **MCP-over-HTTP deferred** — Only stdio transport implemented. HTTP-based MCP transport (for browser/remote clients) deferred.
