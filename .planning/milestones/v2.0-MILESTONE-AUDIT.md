## Architecture Decisions (v2.0)

| Decision | Rationale |
|----------|-----------|
| OTEL as opt-in [otel] extra | Heavy deps; graceful no-op via import guards |
| HTTPXClientInstrumentor global patch | Auto-instruments all httpx clients |
| Custom metrics in lib facade | Where error_class is known; all faces converge |
| In-memory sliding window rate limiter | Zero external deps; single-process suitable |
| mTLS via stdlib ssl.SSLContext | Zero new deps; httpx verify= param |
| OAuth2TokenManager thread-safe cache | Token refreshed before expiry; Bearer per-request |
| Audit logging gated by env var | Zero overhead when disabled |
| UpstreamWSTransport with bounded queue | Backpressure; graphql-transport-ws protocol |
| SSE as StreamingResponse fallback | Works where WebSocket unavailable |
| Copier with Jinja2 parameterization | All hardcoded strings replaced |
