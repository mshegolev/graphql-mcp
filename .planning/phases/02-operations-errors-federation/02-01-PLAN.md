Create `src/graphql_mcp/adapters/outbound/query_guard.py` with two functions:

1. `contains_mutation(query_str: str) -> bool`:
   - Import `from graphql import parse` and `from graphql.language.ast import OperationType`
   - Parse query_str using `parse(query_str)`
   - Iterate `doc.definitions`, check for `hasattr(defn, "operation") and defn.operation == OperationType.MUTATION`
   - Return `True` if any definition is a mutation, `False` otherwise
   - If `parse()` raises any exception (invalid query), return `False` — let the server decide (per RESEARCH.md anti-pattern guidance)
   - Do NOT block subscriptions (`OperationType.SUBSCRIPTION`) — mutation guard only blocks mutations

2. `check_mutation_guard(query_str: str) -> None`:
   - Import `from graphql_mcp.domain.errors import MutationGuardError`
   - Call `contains_mutation(query_str)`
   - If True, raise `MutationGuardError()` (use default message)
   - If False, return None (no-op)

File must start with `from __future__ import annotations`. Export both functions in module scope.

Class `SchemaAnalyzer` with SDL-hash-keyed cache and three methods:

1. `build_summary(self, sdl: str) -> SchemaSummary`:
   - Call `_get_or_build_schema(sdl)` to get cached `GraphQLSchema`
   - Extract `query_fields = list(schema.query_type.fields.keys()) if schema.query_type else []`
   - Map `type_map` to populate `_GRAPHQL_TYPE_MAP`:
     ```python
     _GRAPHQL_TYPE_MAP = {
         GraphQLObjectType: "OBJECT",
         GraphQLInterfaceType: "INTERFACE",
         GraphQLEnumType: "ENUM",
         GraphQLUnionType: "UNION",
         GraphQLInputObjectType: "INPUT_OBJECT",
         GraphQLScalarType: "SCALAR",
     }
     ```
   - Build `types` list: for each `(name, obj)` in `schema.type_map.items()` where `not name.startswith("__")`, create `TypeSummary(name=name, kind=_GRAPHQL_TYPE_MAP.get(type(obj), ""), description=obj.description or "")`
   - Return `SchemaSummary(query_fields=query_fields, types=types)`

2. `describe_type(self, sdl: str, type_name: str) -> TypeInfo | None`:
   - Call `_get_or_build_schema(sdl)`
   - `type_obj = schema.type_map.get(type_name)` — return `None` if not found
   - Determine `kind` from `_GRAPHQL_TYPE_MAP`
   - Build `fields` list: if `hasattr(type_obj, "fields")`, iterate `type_obj.fields.items()` creating `FieldInfo(name=fname, type_str=str(fobj.type), description=fobj.description or "", args=[f"{aname}: {aobj.type}" for aname, aobj in fobj.args.items()] if hasattr(fobj, "args") else [])`
   - Look up subgraph: call `_find_subgraph_for_type(sdl, type_name)` — parse SDL AST, find `@join__type(graph: X)` directive on the type definition, map `X` to subgraph name via `join__Graph` enum. Return None if no directive.
   - Return `TypeInfo(name=type_name, kind=kind, description=type_obj.description or "", fields=fields, subgraph=subgraph)`

3. `list_subgraphs(self, sdl: str) -> list[Subgraph]`:
   - Parse SDL with `parse(sdl)` 
   - Step 1: Find `join__Graph` enum — iterate `doc.definitions` for `EnumTypeDefinitionNode` with `defn.name.value == "join__Graph"`. If not found, return `[]` (not a supergraph).
   - Step 2: Extract subgraph metadata from enum values — for each value, find `@join__graph` directive, extract `name` and `url` from arguments. Build `subgraph_map: dict[str, dict]` keyed by enum value name.
   - Step 3: Validate — require BOTH `join__Graph` enum AND at least one `@join__graph` directive (per RESEARCH.md Pitfall 2)
   - Step 4: Map type ownership — iterate all definitions with `@join__type(graph: X)` directive, append type name to matching subgraph's `owned_types`
   - Return `[Subgraph(name=info["name"], url=info["url"], owned_types=info.get("owned_types", [])) for info in subgraph_map.values()]`

4. Private `_get_or_build_schema(self, sdl: str) -> GraphQLSchema`:
   - Compute `sha = hashlib.sha256(sdl.encode()).hexdigest()`
   - If `self._cache_key == sha`, return `self._cached_schema`
   - Try `build_schema(sdl)` — if it raises (e.g., unknown directives in supergraph), fall back to `build_schema(sdl, assume_valid=True)` or `build_schema(sdl, assume_valid_sdl=True)`. If still fails, re-raise.
   - Store `self._cache_key = sha`, `self._cached_schema = schema`
   - Return schema

5. Private `_find_subgraph_for_type(self, sdl: str, type_name: str) -> str | None`:
   - Extract subgraph enum-to-name mapping from `join__Graph` enum (reuse logic from list_subgraphs)
   - Find the type definition node with matching name and `@join__type(graph: X)` directive
   - Return resolved subgraph name or None

Init: `self._cache_key: str = ""`, `self._cached_schema: Any = None`
