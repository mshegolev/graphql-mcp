**Step 1: Update `src/graphql_mcp/ports/transport.py`** — Add `AsyncGraphQLTransport` protocol below the existing `GraphQLTransport`. Keep the sync protocol untouched. The async protocol has:
- `async def execute(self, query: str, variables: dict[str, Any] | None = None) -> QueryResult`
- `async def post_raw(self, body: dict[str, Any]) -> QueryResult`

Both return the same `QueryResult` type. Use `@runtime_checkable`. Import `Awaitable` is NOT needed — just mark methods `async` in the Protocol body.

**Step 2: Create `src/graphql_mcp/adapters/outbound/async_http_transport.py`** — Mirror the sync `HttpTransport` exactly, but with:
- `httpx.AsyncClient` instead of `httpx.Client`
- `async def execute(...)` and `async def post_raw(...)` instead of sync
- `await self._client.post(...)` instead of `self._client.post(...)`
- `async def close(self)` calling `await self._client.aclose()`
- Same constructor signature: `(endpoint, bearer_token="", timeout=30.0, ssl_verify=True, headers=None, codec=None)`
- Same error handling: catch `(httpx.HTTPError, httpx.StreamError, OSError)` → TRANSPORT error. Non-200 status → TRANSPORT. `(ValueError, TypeError)` from codec.decode → TRANSPORT. GraphQL errors[] → GRAPHQL. Clean response → OK.
- Use `get_codec()` as default codec, same as sync.
- Use `from __future__ import annotations` at top.
